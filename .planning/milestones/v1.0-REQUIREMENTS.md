## v1 Requirements

### Config & Environment (ENV)

- [x] **ENV-01**: Unified analysis config (YAML or JSON) that imports/extends the PRL pick_best_cue task config
- [x] **ENV-02**: Task environment simulator that generates trial-by-trial input sequences
- [x] **ENV-03**: Environment correctly reproduces PRL task structure: 3 sets x (acquisition + N reversals) + transfer phase
- [x] **ENV-04**: Stable vs. volatile phase labeling
- [x] **ENV-05**: RNG seeding: environment generation reproducible from a single integer seed

### Models (MOD)

- [x] **MOD-01**: 2-level binary HGF with 3 parallel input nodes via pyhgf Network API
- [x] **MOD-02**: 3-level binary HGF with 3 parallel input nodes sharing a single volatility parent
- [x] **MOD-03**: Both models accept trial-by-trial binary input with partial feedback
- [x] **MOD-04**: Forward simulation producing belief trajectories
- [x] **MOD-05**: Model parameters exposed as named dict with clear mapping to pyhgf node parameters

### Response Model (RSP)

- [x] **RSP-01**: Custom softmax + stickiness response function compatible with pyhgf's surprise() API
- [x] **RSP-02**: Softmax over 3 cues with inverse temperature and stickiness
- [x] **RSP-03**: Response function parameters: beta (>0) and zeta (any sign)
- [x] **RSP-04**: Response function handles first trial gracefully

### Simulation (SIM)

- [x] **SIM-01**: Agent simulator for complete simulated datasets
- [x] **SIM-02**: Batch simulation: N_per_group x N_groups x N_sessions synthetic datasets
- [x] **SIM-03**: Group parameter specification with (mean, sd) distributions
- [x] **SIM-04**: Session parameter specification via additive deltas
- [x] **SIM-05**: Group x session interaction (different deltas by group)
- [x] **SIM-06**: Tidy output format with all required columns

### Fitting (FIT)

- [x] **FIT-01**: Single-subject fitting via PyMC + pyhgf HGFDistribution
- [x] **FIT-02**: Documented, justified priors for each free parameter
- [x] **FIT-03**: MCMC diagnostics: R-hat, ESS, trace plots saved per fit
- [x] **FIT-04**: Batch fitting with posterior summaries to results DataFrame
- [x] **FIT-05**: Fit both 2-level and 3-level models for model comparison

### Parameter Recovery (REC)

- [x] **REC-01**: Recovery analysis: scatter plots of true vs. recovered parameters
- [x] **REC-02**: Recovery r > 0.7 for omega_2, beta, zeta; omega_3/kappa explicitly documented
- [x] **REC-03**: Confusion/correlation matrix between parameters
- [x] **REC-04**: Recovery run with at least N=30 simulated participants per regime

### Model Comparison (CMP)

- [x] **CMP-01**: Per-subject model evidence via WAIC
- [x] **CMP-02**: Random-effects BMS (Rigoux et al. 2014) with exceedance probabilities
- [x] **CMP-03**: Model comparison stratified by group with protected exceedance probability
- [x] **CMP-04**: Model comparison results summary table and visualization

### Group Analysis (GRP)

- [x] **GRP-01**: Extract point estimates per participant
- [x] **GRP-02**: Mixed-effects model: group x session on each parameter
- [x] **GRP-03**: Phase-stratified analysis: stable vs. volatile (win-stay/lose-shift)
- [x] **GRP-04**: Raincloud/interaction plots by group x session
- [x] **GRP-05**: Effect sizes for all group comparisons

### Interactive GUI (GUI)

- [x] **GUI-01**: Jupyter widget (ipywidgets) running in VSCode
- [x] **GUI-02**: Parameter sliders for omega_2, omega_3, kappa, beta, zeta, mu_1_0, mu_3_0
- [x] **GUI-03**: Real-time plot updates: beliefs, volatility, choice probabilities, learning rate
- [x] **GUI-04**: Ground-truth reward probabilities overlaid on belief trajectories
- [x] **GUI-05**: Toggle between 2-level and 3-level model
- [x] **GUI-06**: Pre-set parameter profiles for quick comparison

### Infrastructure (INF)

- [x] **INF-01**: pyproject.toml with pinned dependencies
- [x] **INF-02**: Reproducible environment with seed-controlled RNG
- [x] **INF-03**: Project structure: src/, notebooks/, configs/, tests/
- [x] **INF-04**: Unit tests for environment, response function, and mini-batch recovery
- [x] **INF-05**: README with setup instructions
