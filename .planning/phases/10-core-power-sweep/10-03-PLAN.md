Create `src/prl_hgf/power/iteration.py` with these components:

**1. `build_arrays_from_sim(subset: pd.DataFrame) -> tuple[np.ndarray, np.ndarray, np.ndarray]`**

Duplicates the logic of `fitting.batch._build_arrays` (do NOT import the private function):
- Takes a DataFrame subset for one participant-session, sorted by trial.
- Reads `cue_chosen` and `reward` columns.
- Returns `(input_data_arr, observed_arr, choices_arr)` with shapes (n_trials, 3), (n_trials, 3), (n_trials,).
- `input_data_arr[t, cue] = reward` and `observed_arr[t, cue] = 1` only for the chosen cue.

**2. `_compute_bms_power(sim_df, idata_3level, idata_2level) -> tuple[float, bool]`**

Private helper implementing Research Pattern 4:
- Get unique (participant_id, group, session) keys from sim_df.
- For each key, filter sim_df to that participant-session, sort by trial.
- Call `build_arrays_from_sim(subset)` to get input/observed/choices arrays.
- For each model ("hgf_3level" then "hgf_2level"), look up idata from the respective dict.
  - If idata is None (failed fit), skip this participant for this model.
  - Otherwise call `compute_subject_waic(input_arr, obs_arr, choices_arr, idata, model_name)`.
  - Collect rows: {participant_id, model, elpd_waic}.
- Build a DataFrame, average elpd_waic across sessions per (participant_id, model).
- Pivot to (n_subjects, 2) matrix with columns [hgf_2level, hgf_3level]. Drop rows with NaN.
- If fewer than 3 subjects remain, return (0.5, False).
- Call `run_group_bms(matrix, ["hgf_2level", "hgf_3level"])`.
- Extract `xp_3level = float(bms_result["xp"][1])`. **Index 1 = hgf_3level** (critical).
- Return `(xp_3level, xp_3level > 0.75)`.

**3. `_extract_diagnostics(sim_df, fit_df) -> tuple[float, int, float]`**

Private helper:
- recovery_r: Call `build_recovery_df(sim_df, fit_df, exclude_flagged=True, min_n=0)` then `compute_recovery_metrics(recovery_df)`. Extract r for "omega_2". Return NaN if omega_2 not found.
- n_divergences: Sum `fit_df["n_divergences"]` if column exists, else 0. (Note: fit_batch doesn't produce this column currently — use 0 as placeholder and document.)
- mean_rhat: `fit_df["r_hat"].mean()` (NaN-safe with `nanmean`).
- Return `(recovery_r, n_divergences, mean_rhat)`.

**4. `run_power_iteration(base_config, n_per_group, effect_size_delta, iteration, child_seed, power_config, n_chains=2, n_draws=500, n_tune=500) -> list[dict]`**

Main orchestrator function:

```
Step 1: Build frozen config
    cfg = make_power_config(base_config, n_per_group, effect_size_delta, child_seed)

Step 2: Simulate
    sim_df = simulate_batch(cfg)

Step 3: Fit 3-level model (primary for BF path)
    fit_df_3, idata_3level = fit_batch(
        sim_df, "hgf_3level", return_idata=True,
        random_seed=child_seed, cores=1,
        n_chains=n_chains, n_draws=n_draws, n_tune=n_tune,
    )

Step 4: Fit 2-level model (for BMS path)
    fit_df_2, idata_2level = fit_batch(
        sim_df, "hgf_2level", return_idata=True,
        random_seed=child_seed + 1, cores=1,
        n_chains=n_chains, n_draws=n_draws, n_tune=n_tune,
    )

Step 5: Compute contrasts + BF (uses fit_df_3 for omega_2 posterior means)
    contrast_results = compute_all_contrasts(
        fit_df_3, parameter="omega_2", bf_threshold=power_config.bf_threshold
    )

Step 6: Compute BMS power
    bms_xp, bms_correct = _compute_bms_power(sim_df, idata_3level, idata_2level)

Step 7: Extract diagnostics
    recovery_r, n_divergences, mean_rhat = _extract_diagnostics(sim_df, fit_df_3)

Step 8: Build 3 result dicts (one per contrast type)
    For each contrast in contrast_results:
        result = {
            "sweep_type": contrast["sweep_type"],
            "effect_size": effect_size_delta,
            "n_per_group": n_per_group,
            "trial_count": cfg.task.n_trials_total,
            "iteration": iteration,
            "parameter": "omega_2",
            "bf_value": contrast["bf_value"],
            "bf_exceeds": contrast["bf_exceeds"],
            "bms_xp": bms_xp,
            "bms_correct": bms_correct,
            "recovery_r": recovery_r,
            "n_divergences": n_divergences,
            "mean_rhat": mean_rhat,
        }

Return list of 3 dicts.
```

Each dict conforms to POWER_SCHEMA (13 columns). The caller (entry point) writes each to a separate parquet file.
