  2. Create `src/prl_hgf/analysis/group.py` with the following functions:

  **build_estimates_wide(fit_df, model, exclude_flagged=True) -> pd.DataFrame**
  - Takes the long-form fit_batch() output DataFrame.
  - Filters to the specified model variant (e.g., "2level" or "3level").
  - If exclude_flagged: drops rows where flagged == True.
  - Pivots using pivot_table: index=["participant_id", "group", "session"], columns="parameter", values="mean".
  - reset_index() to flatten.
  - Returns wide-form DataFrame: one row per (participant_id, group, session), with parameter columns (omega_2, beta, zeta, and for 3-level: omega_3, kappa).

  **fit_group_model(estimates_wide, outcome, draws=2000, tune=1000, chains=4, random_seed=42) -> tuple[bmb.Model, az.InferenceData]**
  - Takes the wide-form estimates DataFrame and one outcome parameter name.
  - CRITICAL: Cast group to category and session to str then category BEFORE building the model. The C() wrapper in the formula alone is NOT sufficient if session is numeric.
  - Formula: `f"{outcome} ~ C(group) * C(session) + (1 | participant_id)"`
  - Build model with bmb.Model(formula, df, dropna=True).
  - Fit with model.fit(draws=draws, tune=tune, chains=chains, random_seed=random_seed).
  - Log the posterior variable names via `list(idata.posterior.data_vars)` at INFO level for debugging.
  - Return (model, idata).

  **extract_posterior_contrasts(idata, group_label="post_concussion", session_labels=None) -> pd.DataFrame**
  - Takes InferenceData from bambi fit.
  - Stacks chain/draw: `posterior = idata.posterior.stack(draws=("chain", "draw"))`.
  - Discovers variable names dynamically by inspecting `list(idata.posterior.data_vars)`.
  - For the group main effect: finds the variable containing the group_label (e.g., `C(group)[T.post_concussion]`).
  - For each session interaction term: finds variables matching `C(group)[T.{group_label}]:C(session)[T.{session}]`.
  - Computes the contrast (group main effect + interaction) for each session.
  - For the reference session (baseline): contrast is just the group main effect.
  - Returns a DataFrame with columns: session, mean, sd, hdi_3, hdi_97.
  - Use az.hdi(contrast_samples, hdi_prob=0.94) for credible intervals.

  **summarize_group_models(estimates_wide, params, **fit_kwargs) -> dict**
  - Convenience function: loops over a list of parameter names, calls fit_group_model for each, calls extract_posterior_contrasts for each, and collects results.
  - Returns dict mapping param name -> {"model": bmb.Model, "idata": InferenceData, "contrasts": DataFrame}.

  Use `from __future__ import annotations` at the top. NumPy-style docstrings. Type hints with Python 3.10+ syntax. Math symbols only inside internal logic; API names are descriptive English.
  </action>

  **compute_cohens_d(estimates_wide, outcome, session, group_a="control", group_b="post_concussion") -> float**
  - Filters estimates_wide to the given session.
  - Extracts outcome values for each group.
  - Uses pingouin.compute_effsize(grp_a_values, grp_b_values, paired=False, eftype="cohen").
  - Returns the float Cohen's d value.

  **compute_effect_sizes_table(estimates_wide, params, sessions) -> pd.DataFrame**
  - Loops over all (parameter, session) combinations.
  - Calls compute_cohens_d for each.
  - Also computes a Bayesian effect size proxy: the posterior mean of the group contrast divided by the pooled SD of the outcome across both groups within that session (simple Bayesian d).
  - Returns DataFrame with columns: parameter, session, cohen_d, n_control, n_post_concussion.
  - The group labels should be discovered dynamically from the "group" column unique values, not hardcoded.
