<objective>
Run the NUTS (PAT-RL) JIT probe on M3 GPU at P ∈ {5, 20} — matched to plan 21-03's
Laplace probe shapes — and populate the NUTS-path rows in the shared
`vb_laplace_vs_nuts_jit.json` + `jit_scaling_sweep.csv`. This is the logp-graph-vs-sampler
isolation test's other half: 21-03 measured Laplace (LBFGS + Hessian, exercises the
logp graph but not NUTS); 21-04 measures NUTS (exercises logp graph AND sampler).
Comparing them names whether the sampler is the marginal bottleneck.

**In scope:**
- New SLURM script `cluster/21_diagnose_nuts.slurm`:
    - Walltime `#SBATCH --time=00:30:00` (SC7).
    - Same GPU allocation, conda env, PATH setup, JAX_COMPILATION_CACHE_DIR as

    - Sets `XLA_FLAGS=--xla_dump_hlo_as_text --xla_dump_to=<per-P archive path>`.
    - Invokes `python scripts/08_run_power_iteration.py --p-scan 5 --diagnostic-mode`
      then `--p-scan 20 --diagnostic-mode` sequentially within one job.
    - After the probes, parses the per-P JAX_LOG_COMPILES logs (which the probe
      archives via a StringIO handler — but for NUTS the probe logs go to
      `cluster/logs/diag21_nuts_<JOBID>.out` not to a dedicated per-P log file,
      since `_run_p_scan_probe` writes CSV not a named log). Follow-up: ADD a NUTS
      log archival step in the SLURM script: after each probe invocation, redirect
      the probe's stderr to `nuts_jax_compiles_P${P}.log`. Details in Task 1 below.

      `nuts_per_p_scaling` top-level key with per-P entries (total_s,

    - Ends with a git auto-push block.

- Produce same-proc warm/cold numbers at P ∈ {5, 20} for the NUTS path that plan

  Step 4 — Post-probe JSON assembly (parse the per-P NUTS logs and merge into
  vb_laplace_vs_nuts_jit.json without clobbering Laplace rows):
    ```bash
    echo "========================================"
    echo "Merging NUTS results into vb_laplace_vs_nuts_jit.json"
    echo "========================================"

    python <<'PYEOF'
    import json
    import re
    from pathlib import Path

    phase_dir = Path(".planning/phases/21-benchmark-bottleneck-diagnosis")
    json_path = phase_dir / "vb_laplace_vs_nuts_jit.json"
    csv_path = phase_dir / "jit_scaling_sweep.csv"

    if json_path.exists():
        with json_path.open("r", encoding="utf-8") as f:
            data = json.load(f)
    else:
        # Should not happen if plan 21-03 ran first; guard anyway.
        data = {
            "per_stage_compile_events": {},
            "per_call_wall_clock": {},
            "per_p_scaling": {},
            "bottleneck_verdict": {
                "bottleneck_layer": "undetermined",
                "evidence": "Filled by plan 21-07.",
            },
            "hlo_op_counts": {},
        }
    # B4: per_stage_compile_times was renamed to per_stage_compile_events
    # (JAX_LOG_COMPILES does not emit timing values); per_call_wall_clock added
    # as a sibling populated from time.perf_counter wrappers in 21-02.

    data.setdefault("nuts_per_p_scaling", {})

    for P in [5, 20]:
        log_path = phase_dir / f"nuts_jax_compiles_P{P}.log"
        if not log_path.exists():
            continue
        text = log_path.read_text(encoding="utf-8")

        compile_events = re.findall(r"Compiling (\S+) for args", text)
        cache_hits = re.findall(r"Persistent compilation cache hit for '(\S+)'", text)
        cache_misses = re.findall(
            r"PERSISTENT COMPILATION CACHE MISS for '(\S+)'", text
        )

        # Extract the cold/warm wall-clock from the probe's own print statement:
        # "--p-scan P complete: status=..., cold=Xs, warm=Ys, row appended to ..."
        match = re.search(
            r"--p-scan\s+\d+\s+complete:.*?cold=([\d.]+)s,\s*warm=([\d.]+)s",
            text,
            re.DOTALL,
        )
        if match:
            cold_s = float(match.group(1))
            warm_s = float(match.group(2))
        else:
            cold_s = None
            warm_s = None

        data["nuts_per_p_scaling"][str(P)] = {
            "cold_s": cold_s,
            "warm_s": warm_s,
            "n_compile_events": len(compile_events),
            "n_cache_hits": len(cache_hits),
            "n_cache_misses": len(cache_misses),
        }

    with json_path.open("w", encoding="utf-8") as f:
        json.dump(data, f, indent=2)

    print(f"Merged NUTS rows for P={list(data['nuts_per_p_scaling'].keys())}")
    PYEOF
    ```
