FUNCTION CONTRACT:

```python
def inspect_network(network: Any) -> dict[str, Any]:
    """Extract topology from a pyhgf Network into a plain dict.

    Parameters
    ----------
    network
        A pyhgf.model.Network instance. Only the following attrs are read:
        ``attributes`` (dict-like indexed by node idx), ``edges``
        (sequence of ``AdjacencyLists`` NamedTuples), ``input_idxs``
        (tuple of int), ``n_nodes`` (int).

    Returns
    -------
    dict
        Keys: ``n_nodes`` (int), ``input_idxs`` (tuple[int, ...]),
        ``nodes`` (tuple[dict, ...] — one entry per unique node index
        reachable from ``input_idxs`` via value_parents / volatility_parents,
        deduplicated by idx).

    Raises
    ------
    ValueError
        If any visited node has a ``node_type`` integer not in
        ``_NODE_TYPE_INT_TO_STR``. Message names the offending node idx,
        the observed integer code, and the full known-codes set.

    Notes
    -----
    Uses BFS from ``input_idxs`` with a ``set[int]`` visited guard so
    that shared volatility parents (e.g. node 6 in the pick_best_cue
    3-level topology, reachable from branches 0/1/2) are emitted
    exactly once.
    """
```

PER-NODE DICT shape (keys — snake_case; no Enum; None where missing):
- idx: int
- kind: str  (mapped via _NODE_TYPE_INT_TO_STR, never the raw int)
- level: int (BFS first-discovery depth; input_idxs are level=1,
  parents of level=k are level=k+1)
- branch_idx: int | None (initial enqueue order for input_idxs; None
  for nodes reachable from multiple branches — this is the "shared
  parent marker")
- value_parents: tuple[int, ...]        (() if edge.value_parents is None)
- volatility_parents: tuple[int, ...]   (() if edge.volatility_parents is None)
- value_children: tuple[int, ...]
- volatility_children: tuple[int, ...]
- tonic_volatility: float | None  (from network.attributes[idx].get(...))
- mean: float | None              (from attributes)
- precision: float | None         (from attributes)

BFS ALGORITHM (use `collections.deque`, `set[int]` for visited):

  1. visited: set[int] = set()
  2. branch_of: dict[int, int | None] = {}
  3. queue = deque()
  4. For branch_idx, input_idx in enumerate(network.input_idxs):
       queue.append((input_idx, 1, branch_idx))
       branch_of[input_idx] = branch_idx
  5. While queue:
       idx, level, branch = queue.popleft()
       if idx in visited: continue
       visited.add(idx)
       edge = network.edges[idx]
       if edge.node_type not in _NODE_TYPE_INT_TO_STR:
           raise ValueError(
               f"Unknown pyhgf node_type code {edge.node_type} at node "
               f"index {idx}. Known codes: {sorted(_NODE_TYPE_INT_TO_STR)}"
           )
       kind = _NODE_TYPE_INT_TO_STR[edge.node_type]
       attrs = network.attributes[idx]
       Emit node dict (fields per shape above).
       For each parent_idx in (edge.value_parents or ()) + (edge.volatility_parents or ()):
         if parent_idx in visited: continue
         prev_branch = branch_of.get(parent_idx)  # None sentinel on first sight
         if parent_idx not in branch_of:
             branch_of[parent_idx] = branch
             new_branch = branch
         elif prev_branch == branch:
             new_branch = branch
         else:
             branch_of[parent_idx] = None   # contested => shared node
             new_branch = None
         queue.append((parent_idx, level + 1, new_branch))
  6. Return {"n_nodes": int(network.n_nodes),
             "input_idxs": tuple(network.input_idxs),
             "nodes": tuple(node_rows)}
     Note: node_rows order is BFS discovery order. Tests assert count,
     not order, but preserving BFS order keeps renders deterministic.
