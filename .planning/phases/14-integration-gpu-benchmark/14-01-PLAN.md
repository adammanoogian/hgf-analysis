  **1. Add `_idata_to_fit_df` helper (place above `run_sbf_iteration`):**
  - Signature: `_idata_to_fit_df(idata: az.InferenceData, model_name: str) -> pd.DataFrame`
  - Extract participant metadata from `idata.posterior.coords`: `participant` (IDs), `participant_group`, `participant_session`.
  - For each parameter variable in `idata.posterior` (skip `participant`, `participant_group`, `participant_session` coords; include all data_vars), for each participant index `i`:
    - Extract posterior draws: `idata.posterior[param].isel(participant=i).values` (shape: chain x draw) then flatten.
    - Compute: mean, sd (np.std), r_hat via `az.rhat(idata.posterior[param].isel(participant=i))`, ess_bulk via `az.ess(idata.posterior[param].isel(participant=i))`, hdi_3%/hdi_97% via `az.hdi(idata.posterior[param].isel(participant=i), hdi_prob=0.94)`.
    - Flag if `r_hat > 1.05` or `ess < 400`.
    - Build row: `participant_id, group, session, model, parameter, mean, sd, hdi_3%, hdi_97%, r_hat, ess, flagged`.
  - Return `pd.DataFrame(rows)`.
  - **Important:** Read participant ordering from `idata.posterior.coords["participant"].values` -- do NOT assume it matches input DataFrame order.
  - **Important:** Use `arviz` functions for r_hat and ess rather than hand-rolling. Import `arviz as az` at the top of the function (deferred import to avoid heavy import at module load).
  - **Gotcha:** `az.rhat` and `az.ess` expect xarray DataArrays. Use `idata.posterior[param].sel(participant=pid)` to get a DataArray with (chain, draw) dims, then pass to `az.rhat()` and `az.ess()`. These return scalar xarray objects -- use `.item()` or `float()` to extract.

  **2. Add `_split_idata` helper:**
  - Signature: `_split_idata(joint_idata: az.InferenceData, participant_idx: int) -> az.InferenceData`
  - Slice: `posterior_slice = joint_idata.posterior.isel(participant=participant_idx)`
  - Return: `az.InferenceData(posterior=posterior_slice)`
  - This produces per-participant InferenceData for `compute_subject_waic`.

  **3. Add `_build_idata_dict` helper:**
  - Signature: `_build_idata_dict(joint_idata: az.InferenceData, participant_ids: list[str], participant_groups: list[str], participant_sessions: list[str]) -> dict[tuple, az.InferenceData]`
  - Read coords from `joint_idata.posterior`: get `participant` coord values, `participant_group` coord values, `participant_session` coord values.
  - Build dict: for each index `i`, key = `(participant_ids[i], participant_groups[i], participant_sessions[i])`, value = `_split_idata(joint_idata, i)`.
  - Return the dict (compatible with `_compute_waic_table` and `_compute_bms_power`).

  **4. Modify `run_sbf_iteration`:**
  - Add parameter: `use_legacy: bool = False` after `sampler`.
  - Keep the existing code as the `if use_legacy:` branch (move Steps 3-4 fit_batch calls into this branch).
  - Add new `else:` branch that:
    a. Imports `fit_batch_hierarchical` from `prl_hgf.fitting.hierarchical` (deferred import inside else block).
    b. Calls `idata_3 = fit_batch_hierarchical(sim_df, "hgf_3level", n_chains=n_chains, n_draws=n_draws, n_tune=n_tune, target_accept=0.9, random_seed=child_seed, sampler=sampler, progressbar=False)`.
    c. Calls `idata_2 = fit_batch_hierarchical(sim_df, "hgf_2level", n_chains=n_chains, n_draws=n_draws, n_tune=n_tune, target_accept=0.9, random_seed=child_seed + 1, sampler=sampler, progressbar=False)`.
    d. Converts to DataFrames: `fit_df_3 = _idata_to_fit_df(idata_3, "hgf_3level")` and `fit_df_2 = _idata_to_fit_df(idata_2, "hgf_2level")`.
    e. Builds idata dicts: `idata_3level = _build_idata_dict(idata_3, ...)` and `idata_2level = _build_idata_dict(idata_2, ...)`.
    f. Metadata lists come from `idata_3.posterior.coords["participant"].values.tolist()`, `idata_3.posterior.coords["participant_group"].values.tolist()`, `idata_3.posterior.coords["participant_session"].values.tolist()`.
  - After the if/else, the rest of the function (WAIC, subsampling, BF, diagnostics) is unchanged -- it uses `fit_df_3`, `fit_df_2`, `idata_3level`, `idata_2level` which are set by either branch.
  - Update the docstring to document `use_legacy`.

  **5. Update imports at top of iteration.py:**
  - Add `from __future__ import annotations` (already present).
  - Do NOT add a module-level import for `fit_batch_hierarchical` -- keep it as a deferred import inside the `else` block to avoid heavy JAX imports when using legacy path.
