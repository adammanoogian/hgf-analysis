   - `from __future__ import annotations`
   - Re-export `make_power_config`, `PowerConfig`, `load_power_config` from `.config`
   - Add `__all__` listing these names

2. Create `src/prl_hgf/power/config.py`:
   a. `PowerConfig` frozen dataclass with fields:
      - `n_per_group_grid: list[int]`
      - `effect_size_grid: list[float]`
      - `n_iterations: int`
      - `master_seed: int`
      - `n_jobs: int`
      - `bf_threshold: float`
      - `__post_init__` validates: n_iterations >= 1, n_jobs >= 1, bf_threshold > 0
   
   b. `load_power_config(path: Path | None = None) -> PowerConfig`:
      - Opens YAML, reads only the `power:` top-level key
      - Raises `ValueError` if `power:` key missing
      - Constructs and returns PowerConfig
      - Default path from `config.CONFIGS_DIR / "prl_analysis.yaml"`
   
   c. `make_power_config(base: AnalysisConfig, n_per_group: int, effect_size_delta: float, master_seed: int) -> AnalysisConfig`:
      - Uses `dataclasses.replace` bottom-up:
        - For each group in `base.simulation.session_deltas.items()`:
          - If group == "psilocybin": `dataclasses.replace(sess, omega_2_deltas=[d + effect_size_delta for d in sess.omega_2_deltas])`
          - Else: keep session config unchanged
        - Build new dict `new_deltas` (new dict, not mutated original)
        - `new_sim = dataclasses.replace(sim, n_participants_per_group=n_per_group, master_seed=master_seed, session_deltas=new_deltas)`
        - `return dataclasses.replace(base, simulation=new_sim)`
      - Must NOT import yaml, read files, or do any I/O
      - Docstring: NumPy-style, describes all parameters and return value

   Follow conventions: `from __future__ import annotations`, type hints with `list[int]` not `List[int]`, line length 88, absolute imports only.

3. Append `power:` section to end of `configs/prl_analysis.yaml`:
   ```yaml
   # ---------------------------------------------------------------------------
   # Power analysis parameters (BFDA)
   # ---------------------------------------------------------------------------
   power:
     n_per_group_grid: [10, 15, 20, 25, 30, 40, 50]
     effect_size_grid: [0.0, 0.3, 0.5, 0.8, 1.0, 1.5]
     n_iterations: 100
     master_seed: 20240101
     n_jobs: 100
     bf_threshold: 10.0
   ```

4. Add `pyarrow>=14.0` to:
   - `pyproject.toml` dependencies list (after `pingouin`)
   - `environment_gpu.yml` pip block (after `pingouin`)
   
   Also add `"pyarrow.*"` to the mypy overrides ignore_missing_imports list in pyproject.toml.

IMPORTANT: Do NOT modify `load_config()` in task_config.py. The existing loader ignores unknown top-level YAML keys, so the `power:` section won't break it. PowerConfig loading is completely separate.
