<objective>
Create SeedSequence utilities, parquet schema enforcement, SLURM array template,
and the entry point script for power sweep iterations.

Purpose: Each SLURM array task needs an independent RNG stream (SeedSequence),
must write results in a fixed 13-column parquet schema, and must be launched by
a SLURM array template with %50 throttle. The entry point script ties everything
together: it reads the task ID, builds an RNG, constructs a config override, and
writes a placeholder parquet row (the actual simulation/fit logic comes in Phase 10).

Output: seeds.py, schema.py, SLURM template, entry point script, unit tests.
</objective>

   Main logic:
   ```python
   def main() -> None:
       args = parse_args()
       
       base_config = load_config()
       power_config = load_power_config()
       
       # Build independent RNG for this task
       rng = make_child_rng(
           master_seed=power_config.master_seed,
           n_jobs=power_config.n_jobs,
           job_index=args.task_id,
       )
       
       # Output path: results/power/job_<JOB_ID>_task_<TASK_ID>.parquet
       output_dir = RESULTS_DIR / "power"
       output_path = output_dir / f"job_{args.job_id}_task_{args.task_id:04d}.parquet"
       
       if args.dry_run:
           # Write placeholder row to verify infrastructure
           placeholder = {
               "sweep_type": "smoke_test",
               "effect_size": 0.0,
               "n_per_group": base_config.simulation.n_participants_per_group,
               "trial_count": base_config.task.n_trials_total,
               "iteration": args.task_id,
               "parameter": "omega_2",
               "bf_value": 1.0,
               "bf_exceeds": False,
               "bms_xp": 0.5,
               "bms_correct": False,
               "recovery_r": 0.0,
               "n_divergences": 0,
               "mean_rhat": 1.0,
           }
           write_parquet_row(placeholder, output_path)
           print(f"Dry run: wrote placeholder to {output_path}")
           return
       
       # Full pipeline will be implemented in Phase 10
       print(f"Task {args.task_id}: full pipeline not yet implemented")
       print(f"RNG state ready, output would go to {output_path}")
   ```
   
   `if __name__ == "__main__": main()`

The script is designed so that `--dry-run` exercises the full infrastructure path (config loading, SeedSequence, parquet write) without needing the simulation/fitting pipeline. This enables the SLURM smoke test in Phase 8.
